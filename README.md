# Amazon Clone – Frontend

A simple **Amazon**-style landing page built using pure HTML and CSS.  
This project focuses on recreating the basic layout, navbar, hero section, and product grid of the Amazon homepage.

---

## Demo

Open the `amazon.html` file in your browser to view the UI.

```bash
# Clone the repository
git clone https://github.com/Sandeeprdy1729/amazon_clone-Frontend-.git

cd amazon_clone-Frontend-

# Open in browser (example for VS Code)
# Right-click amazon.html → "Open with Live Server"
